1. 암송 시스템 고도화: "구절 주소 포함"

    방식 : 암송 화면 상단에 구절 주소(예: 요한복음 3:16)와 본문을 함께 띄워주고, 사용자가 마이크로 말할 때 **"요한복음 삼 장 십육 절..."**이라고 시작해야만 점수가 인식되도록 합니다.

    UI 변경:

        합치기: 기존에 별도로 떨어져 있던 '구절 주소(예: 요한복음 3:16)'를 아래 '본문' 영역의 맨 앞에 합쳐서 하나의 커다란 텍스트 덩어리로 보여줍니다.
        (변경 전) 주소: 요한복음 3:16 / 본문: 하나님이 세상을...
        (변경 후) 본문 영역: 요한복음 3장 16절 하나님이 세상을...
        안내문 배치: 이전에 주소가 있던 자리(상단 서브타이틀)에는 암송 방법을 알려주는 안내문이 들어갑니다.

        예: "💡 말씀 구절을 처음에 말하시고 이어서 말씀을 암송해주세요."

        인식(인공지능) 로직:

        사용자가 암송을 시작할 때 반드시 구절 주소부터 소리 내어 읽어야 합니다.
        주소 부분의 인식 정확도가 낮으면 본문으로 넘어가기 전에 다시 시도하게 하거나 점수를 대폭 차감하여, 주소 외우기의 중요성을 강조합니다.

    채점 : 주소 부분은 한 글자만 틀려도 점수가 크게 깎이도록 엄격하게 설정하거나, 주소를 먼저 말한 후 본문으로 넘어가게 하는 '단계별 인식' 방식을 고려할 수 있습니다.

2. 메인 화면의 변화 : "사용자 유형 선택"
    첫 화면: 앱을 켜자마자 두 개의 큰 카드가 나타납니다.
    A. 외국인 학생 (Foreign Student): "미션 수행 및 보상 받기"
    B. 한국인 사용자 (Korean): "오늘의 말씀 암송하기 (보상 없음)"
    인식 방법: 사용자에게 직접 선택하게 하는 것이 가장 명확합니다. ("당신의 유형을 선택해 주세요")

3. 유형별 상세 프로세스 (UX Flow)
    [외국인 유형] (Reward Mode)
        로그인/회원가입 : 기존 방식 유지 (보상 관리를 위해 필수)
        학번 확인 : 로그인 정보에서 자동 호출
        암송 : 85점 이상 시 통과
        보상 : 라면 선택 및 수령 (기록 저장)
    [한국인 유형] (Praise Mode)
        간편 입장 : 로그인 생략. 이름이나 학번(또는 핸드폰 번호)만 입력 후 즉시 시작.
        암송 : 점수 제한 없음 (시도 자체에 의미)
        칭찬 : "참 잘하셨습니다!", "오늘도 승리하세요!" 같은 화려한 이펙트와 응원 메시지 노출 (보상 선택 화면 생략)

4. 신규 메뉴: "말씀 아카이브 (Archive)"
    위치 : 암송 화면 구석에 작은 ‘달력’ 또는 ‘리스트’ 아이콘 배치.
    내용 : 최근 7일 혹은 한 달간의 말씀을 리스트로 보여주고, 클릭하면 해당 본문을 다시 읽거나 들어볼 수 있게 합니다. (단, 지나간 말씀에 대해서는 보상을 주지 않는 규칙 설정 가능)

5. 관리자 페이지 확장 (Admin)
    통계: 오늘 접속자 중 '외국인'은 몇 명이고 '한국인'은 몇 명인지 비율 확인.
    기록: 한국인 사용자의 경우 '라면 수령' 대신 '암송 완료 횟수'나 '최고 점수' 위주로 리스트업.
